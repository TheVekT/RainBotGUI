# RainBotGUI

[![Python](https://img.shields.io/badge/Python-3.10+-blue.svg)](https://www.python.org/)
[![PyQt6](https://img.shields.io/badge/GUI-PyQt6-green.svg)](https://pypi.org/project/PyQt6/)
[![Status](https://img.shields.io/badge/Status-Archived-red.svg)](#archive-notice)

**RainBotGUI** is a modern desktop controller application built in Python using **PyQt6** and **asyncio**. It acts as a remote graphical administrative dashboard for **RainBot** (a companion Discord bot) using a real-time WebSocket protocol.

---

>

## Key Features

### Real-Time WebSocket Connection
*   Quick settings configurations via a localized `settings.json` file.
*   Interactive dashboard to connect, disconnect, and monitor connection status with visual connection state indicators (neon-glow indicators and custom state icons).

### Interactive Remote Terminal
*   **Slash Command execution**: Send administrative commands (e.g. `/disconnect`, server control) directly to the bot.
*   **Command History**: Quick navigation through previously entered commands using the **Up/Down arrow keys**.
*   **Autocompletion Tooltip**: View all registered commands directly inside the help tooltip.
*   **Log Streaming**: Terminal output prints real-time logs streamed from the bot server.
*   **Search**: Built-in search functionality to quickly find text within the terminal window.

### Log Manager & Viewer
*   Filter, select, and view active and historical log categories (e.g., info, errors, warnings).
*   Search within log files using a matching query filter.
*   Download and save log content directly to local `.txt` files.

### Discord Member Activity & Analytics
*   **Live Member List**: Displays real-time Discord statuses (online, idle, dnd, offline), usernames, and avatars.
*   **Voice Tracking Statistics**: Detailed metrics for Discord users including:
    *   Voice channel time (Voice Online)
    *   Muted/Deafened duration
    *   Offline/Online duration
    *   Time spent alone in a voice channel (Voice Alone)
    *   AFK duration
*   **Interactive Calendar**: Select specific dates to fetch detailed user stats from that day.
*   **Zoomable Timeline Charts**: Visualize daily activity patterns using high-quality timeline graphics rendered with `PyQt6.QtCharts` (supports double-click to zoom into specific events).

### Modern UI/UX
*   **Frameless Window**: Custom titlebar dragging and resizing support utilizing native Windows APIs (`ctypes`, `win32` window messages).
*   **Collapsible Sidebar Menu**: Animated, space-saving sidebar with custom-designed icon hover states.
*   **Smooth Tab Transitions**: Dynamic fade-in animations when switching views.
*   **JetBrains Mono Typography**: High-readability font system for a developer-oriented look.

## Tech Stack & Dependencies

*   **Language**: [Python](https://www.python.org/) (3.10+)
*   **GUI Framework**: [PyQt6](https://pypi.org/project/PyQt6/) (v6.11.0+) & [PyQt6-Charts](https://pypi.org/project/PyQt6-Charts/) (v6.11.0+)
*   **Async Event Loop**: [qasync](https://github.com/CalebCassidy/qasync) (v0.28.0+)
*   **Networking Protocol**: [websockets](https://websockets.readthedocs.io/) (v16.0+)
